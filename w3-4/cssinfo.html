CSS 소개 영상

CSS

종속형 시트 또는 캐스케이딩 스타일 시트(Cascading Style Sheets, CSS)는 마크업 언어가 실제 표시되는 방법을 기술하는 언어로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다. W3C의 표준이며, 레이아웃과 스타일을 정의할 때의 자유도가 높다. 마크업 언어가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리 같은 꾸미는 역할을 담당한다고 할 수 있다. 즉, HTML 구조는 그대로 두고 CSS 파일만 변경해도 전혀 다른 웹사이트처럼 꾸밀 수 있다. 현재 개발 중인 CSS3의 경우 그림자 효과, 그라데이션, 변형 등 그래픽 편집 프로그램으로 제작한 이미지를 대체할 수 있는 기능이 추가되었다. 또한 다양한 애니메이션 기능이 추가되어 어도비 플래시를 어느 정도 대체하고 있다.

Selector

CSS에서 Selector는 마크업 언어 자체의 태그와 속성을 일치시켜서 마크업 언어의 어느 부분에 스타일을 적용할 지 선언한다. 엘리먼트, 아이디, 클래스, 특정 타입의 엘리먼트 등등 다양한 조건을 붙여서 적용이 가능하다.


Pseudo Selector

Pseudo Selector는 가상선택자로 실제로는 존재하지 않지만 특정 엘리먼트에 조건을 붙여서 해당 조건을 만족할 때 효과를 주는 방식이다. 예를 들면 이 페이지의 상단에 존재하는 네비게이션 바는 마우스가 hover 상태가 되면 색이 변화한다. 이때 hover는 가상선택자로 설정하여 사용한다.

SASS / SCSS

“Sass는 문서의 구조를 깔끔하고 구조적으로 기술하는 데 사용하는 기술인 css의 상위에 있는 메타언어로, 자바스크립트처럼 특정 속성의 값을 변수로 선언하여 필요한 곳에 선언된 변수를 적용할 수도 있고, 반복되는 코드를 한 번의 선언으로 여러 곳에서 재사용할 수 있도록 도와주는 일종의 CSS 전처리기(pre-processor)이다.” 따라서, 우리가 사용하는 일반 언어처럼 재사용성을 높이고 활용성을 높여주는 역할을 한다. 일종의 SuperSet으로 보아도 무방하다.

HTML의 템플릿 엔진처럼 CSS내부에 프로그래밍 언어와 비슷하게 반복문이나 조건문 그리고 함수나 변수 등등을 설정하여 중복되어 늘어나는 일을 방지하고 한 번 작성한 CSS를 다른 곳에 재사용할 수 있게 재사용성을 높여준다.

CSS 활용

CSS는 쉬우면서도 명확하게 상호관계를 이해하지 못하면 자기가 원하는대로 만드는 것이 어렵다. 이를 위해 대부분 웹페이지를 제작하는 경우 마크업과 CSS를 전담하는 디자이너가 따로 존재하며 사내에서 직접 CSS 라이브러리를 제작하거나 그렇지 않은 경우 오픈소스로 존재하는 라이브러리를 많이 사용한다. 대표적인 예가 바로 지금 이 페이지에 적용되어 있는 BootStrap으로 CDN으로 간단히 연결하여 사용할 수 있기도 하며 jQuery와 연동되어 각종 애니메이션도 같이 지원한다. 아래와 같이 라이브러리를 이용하거나 직접 CSS를 작성하여 사용할 수 있다.
